=============================
Configuring Aqsis for Houdini
=============================

Aqsis can be defined as the default RenderMan renderer using the following environment variables:

   +------------------------------+--------------------------------+
   | Name                         | Value                          |
   +==============================+================================+
   | HOUDINI_DEFAULT_RIB_RENDERER | |HOUDINI_DEFAULT_RIB_RENDERER| |
   +------------------------------+--------------------------------+
   | HOUDINI_VIEW_RMAN            | aqsis                          |
   +------------------------------+--------------------------------+
