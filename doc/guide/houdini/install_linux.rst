=====================================
Installing Aqsis for Houdini on Linux
=====================================

Houdini integration is handled through the `Neqsus`_ plugin, distributed with Aqsis Renderer.


Fedora/Debian/Ubuntu
====================

1. Copy contents of ``/usr/share/aqsis/plugins/houdini`` to::

   $HOME\|HOUDINI_PATH|

2. Execute ``post.sh`` from above location, and follow on-screen instructions.

.. note::
   +------------------------------+--------------------------------+
   | Environment Variable         | Example                        |
   +==============================+================================+
   | HOME                         | /home/                         |
   +------------------------------+--------------------------------+
