=======================================
Installing Aqsis for Houdini on Windows
=======================================

Houdini integration is handled through the `Neqsus`_ plugin, distributed with Aqsis Renderer.


Windows XP/Vista/7
==================

1. Copy contents of ``%AQSISHOME%\plugins\houdini`` to::

   %USERPROFILE%\My Documents\|HOUDINI_PATH|.0

2. Execute ``post.bat`` from above location, and follow on-screen instructions.

.. note::
   +------------------------------+--------------------------------+
   | Environment Variable         | Example                        |
   +==============================+================================+
   | AQSISHOME                    | C:\\Program Files\\Aqsis       |
   +------------------------------+--------------------------------+
   | USERPROFILE                  | C:\\Users\\                    |
   +------------------------------+--------------------------------+
